#### Using

Just call MobileSdk.bindNewCardForResult(...) //for bankcard binding 
          MobileSdk.paymentForResult(...) //for payment
          MobileSdk.paymentForResult(saveCard, ...) //for saved card payment 

And wait for result from onActivityResult(...)
